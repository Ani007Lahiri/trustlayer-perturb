# External-receipt hand-off kit

These four steps carry the last Impact points (live PyPI package, an open OpenProblems PR,
a citable DOI, and real adopter outreach). They require YOUR accounts/credentials, so I've
prepared everything ready-to-execute rather than doing them for you. Est. ~30 min total.

1. `1_PYPI_PUBLISH.sh`     — build + upload the pip package; verify a clean install. (~10 min)
2. `2_OPENPROBLEMS_PR.md`  — branch + PR title + paste-ready PR body + which files to include. (~10 min)
3. `3_ZENODO_DOI.md`       — Zenodo deposition metadata for an instant DOI. (~5 min)
4. `4_OUTREACH_DRAFTS.md`  — two emails to send from your account. Send early so a reply can
                             land before 9PM. Only quote interest that actually arrives. (~5 min)

Everything references artifacts already saved: the package (release/trustlayer/), the
OpenProblems component (release/openproblems_pr/), the release tarball, and the preprint
abstract.
